# 4.2 傳送一般資料

請在 `practice` 資料夾下，建立 **`fetch.html`**，測試以下原始碼：

```html
<!DOCTYPE html>
<html lang="zh-Hant">
  <head>
    <meta charset="utf-8">
    <title>Fetch</title>
  </head>
  <body>
    <h1>fetch 語法：GET</h1>
    
    <script>

      var fetch_url = "https://notes.webmix.cc/ajax/teach/simple_ajax.php";
      
      fetch(fetch_url).then((res) => {
        if(res.ok){
          return res.json();
        }
      }).then((data) => {
        console.log(data);
      });
    </script>
  </body>
</html>
```
