# 3.4 認識 HTTP Verb

## 認識 HTTP Verb(動詞)

也稱做 request method(請求方式)。

參考：[https://developer.mozilla.org/en-US/docs/Web/HTTP/Methods](https://developer.mozilla.org/en-US/docs/Web/HTTP/Methods)

RESTful API 針對資源(resource)的操作：

* GET：取得資料。
* POST：新增資料。
* PUT：整筆資料更新。
* DELETE：刪除資料。
* PATCH：部份資料更新。(目的：節省頻寬)
